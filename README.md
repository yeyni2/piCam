## To run the app you need to clone
1. picamera 
2. dlib

<u>**dlib:**</u>

``` commandline
git clone https://github.com/davisking/dlib.git
cd dlib
mkdir build
cd build
cmake ..
cmake --build .
cd ..
python setup.py install
```

<u>**picamera:**</u>

```commandline
git clone https://github.com/waveform80/picamera.git
cd picamera
python setup.py install
```
